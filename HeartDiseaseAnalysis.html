Heart Disease Dataset Analysis

Overview

This project focuses on analyzing a heart disease dataset using various machine learning models to predict the presence or absence of heart disease. The analysis involves data preprocessing, model development, and evaluation to identify the best predictive model.

Project Introduction

The Heart Disease Dataset Analysis project aims to predict heart disease using several attributes such as age, sex, chest pain type, resting blood pressure, and cholesterol levels. The project includes data cleaning, feature selection, model training, and evaluation to achieve accurate predictions.

Packages Used

  • Pandas: For data manipulation and analysis.
  • NumPy: For scientific computing with support for arrays and matrices.
  • Seaborn: For data visualization based on Matplotlib.
  • Matplotlib: For creating static, animated, and interactive visualizations.
  • Scikit-learn: For machine learning algorithms and model evaluation.

Project Steps Followed

  • Dataset Import: Loaded the dataset using Pandas and explored its structure.
  • Data Cleansing: Checked for missing values and handled them appropriately.
  • Feature Selection: Selected relevant features for model training.
  • Data Preprocessing: Standardized the data for model compatibility.
  • Model Development: Developed models using Linear Regression, Decision Tree Regressor, and Neural Network Regressor.
  • Model Evaluation: Evaluated models using MAE, RMSE, and R² score.
  • Visualization: Created scatter plots for actual vs. predicted values.

Observations

Based on the evaluation metrics of the different models, the following observations were made:

  • Linear Regression:
    • R²: 0.37
    • MAE: 14.13
    • RMSE: 18.54
    • Accuracy: 36.61%
  • Decision Tree Regressor:
    • R²: 0.94
    • MAE: 0.80
    • RMSE: 5.58
    • Accuracy: 94.27%
  • Neural Network Regressor:
    • R²: 0.34
    • MAE: 9.49
    • RMSE: 18.87
    • Accuracy: 34.32%

The Decision Tree Regressor performed the best among the models evaluated, with the highest R² accuracy and an accuracy percentage of 94.27%. The Linear Regression and Neural Network Regressor models showed significantly lower performance in comparison.
