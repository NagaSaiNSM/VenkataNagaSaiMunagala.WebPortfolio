YouTube Data Analysis

Overview

This project involves analyzing YouTube data to extract meaningful insights from user comments. By performing sentiment analysis, generating word clouds, and analyzing emoji usage, this project aims to understand the nature of interactions on YouTube videos. The project utilizes various Python libraries for data manipulation, visualization, and sentiment analysis.

Project Introduction

The YouTube Data Analysis project is designed to analyze comments from YouTube videos to derive insights into user sentiments, popular words, and frequently used emojis. The analysis involves cleaning the data, performing sentiment analysis using TextBlob, generating word clouds for positive and negative comments, and analyzing the frequency and usage of emojis. The project also involves handling and processing large datasets efficiently.

Packages Used

  • Pandas: For data manipulation and analysis, providing data structures and operations for manipulating numerical tables and time series.
  • NumPy: A fundamental package for scientific computing with Python, providing support for arrays and matrices, along with mathematical functions to operate on these data structures.
  • Seaborn: A data visualization library based on Matplotlib, providing a high-level interface for drawing attractive and informative statistical graphics.
  • Matplotlib: A plotting library for the Python programming language and its numerical mathematics extension NumPy. It provides an object-oriented API for embedding plots into applications.
  • TextBlob: A library for processing textual data, providing a simple API for diving into common natural language processing (NLP) tasks such as part-of-speech tagging, noun phrase extraction, sentiment analysis, classification, translation, and more.
  • WordCloud: A library for generating word clouds, which are visual representations of text data where the size of each word indicates its frequency or importance.
  • emoji: A library for handling and processing emojis in text data, allowing for analysis of emoji usage frequency.
  • Plotly: A graphing library that makes interactive, publication-quality graphs online. It can be used to create various types of charts and visualizations.

Project Steps Followed

  • Dataset Import: Imported the dataset using Pandas and displayed the shape and initial rows of the dataset for a preliminary overview.
  • Data Cleansing: Checked for and handled missing values by removing rows with null values.
  • Sentiment Analysis: Utilized TextBlob to perform sentiment analysis on the comment texts. Calculated the polarity of each comment and added it as a new column in the dataset.
  • Word Cloud Generation: Filtered comments based on their polarity (positive and negative). Generated word clouds for both positive and negative comments using the WordCloud library.
  • Emoji Analysis: Extracted and counted the frequency of emojis used in the comments. Visualized the most common emojis using Plotly.
  • Data Collection and Merging: Combined additional data files from various regions into a single dataframe. Removed duplicate rows and exported the cleaned dataset to CSV, JSON, and SQLite database formats.
  • Category Analysis: Analyzed the most liked categories of videos based on the cleaned and merged dataset. Mapped category IDs to their respective names using additional JSON data.
  • Visualization: Created various plots and visualizations to represent the findings and insights from the analysis.
